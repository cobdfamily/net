# net

Small Node demo using `@cobd/bluefin` — iterates UI elements, reads
roles, and wires the bluefin event bus to stdout. Handy as a
scratchpad while working on the Tuna accessibility ecosystem.

## Run

```
npm install
node index.js     # battery + active element demo
node cli.js       # TF cursor / traversal demo
```
